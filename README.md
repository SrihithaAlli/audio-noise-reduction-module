# Audio Noise Reduction Module

## Overview

This project is an audio preprocessing module developed as part of an AI/ML internship project.

The module reduces background noise from audio recordings while preserving speech clarity, making the audio more suitable for speech-to-text transcription systems.

## Features

* Load WAV audio files
* Convert stereo audio to mono
* Reduce background noise
* Save cleaned audio output

## Technologies Used

* Python
* NumPy
* SoundFile
* NoiseReduce

## Workflow

Input Audio → Noise Reduction → Cleaned Audio

## Files

* input.wav : Original audio file
* cleaned.wav : Noise-reduced audio file
* noise_reduction.py : Main Python implementation

## Objective

Reduce background noise before transcription while preserving voice clarity.
